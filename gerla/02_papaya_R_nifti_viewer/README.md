# Interactive exploration of MRI images in R

LC feb 2025

The great [John Muschelli](https://johnmuschelli.com/) has created a fantastic interactive R wrapper called [papayawidget](https://johnmuschelli.com/papayaWidget/), which allows you to include a [Papaya](https://rii-mango.github.io/Papaya/) viewer in an RMarkdown notebook, and to export them as a single-page HTML. This is _not_ just an image viewer, rather it implements a full-fledged interactive viewer in the notebook!

To follow this tut you need to have R with RMarkdown installed - and of course the papayawidget library as well as the RNifti library. See instructions in the `papaya_TUT.Rmd`.

Otherwise you can just download the `papaya_TUT.html` and open it in a browser.
